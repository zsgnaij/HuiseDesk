# AI 对话界面预览

```
┌─────────────────────────────────────────────────────┐
│  💬 AI 对话助手                    [清空对话]      │ ← 头部
├─────────────────────────────────────────────────────┤
│                                                     │
│  ┌──────────────────────────────┐                 │
│  │ 🤖 AI                  10:30 │                 │
│  │ 你好！我是 AI 助手...         │                 │
│  └──────────────────────────────┘                 │
│                                                     │
│                  ┌──────────────────────────────┐  │
│                  │ 👤 你              10:31     │  │
│                  │ 你好，请介绍一下自己          │  │
│                  └──────────────────────────────┘  │
│                                                     │
│  ┌──────────────────────────────┐                 │
│  │ 🤖 AI                  10:31 │                 │
│  │ 我是基于 DeepSeek 的...      │                 │ ← 消息区域
│  │ （流式显示中...）▌           │                 │   （可滚动）

│                                                     │

├─────────────────────────────────────────────────────┤

│ └─────────────────────────────────────────────┘   │
│ 42 字符                          [取消] [发送]    │

```

## 颜色方案

### 默认主题

- **背景色**: #f5f7fa (浅灰蓝)
- **用户消息**: #2196F3 (蓝色气泡，白色文字)
- **AI 消息**: #FFFFFF (白色气泡，深灰文字)
- **发送按钮**: #4CAF50 (绿色)
- **取消按钮**: #ff9800 (橙色)
- **清空按钮**: #f44336 (红色)

### 状态颜色

- **加载中**: 按钮变灰，显示"发送中..."
- **错误提示**: 红色背景 (#ffebee)，红色文字 (#c62828)

## 交互效果

### 发送消息流程

```
1. 用户输入文字
   ↓
2. 按 Enter (或点击发送按钮)
   ↓
3. 用户消息立即显示（蓝色气泡，右对齐）
   ↓
4. 创建空的 AI 消息气泡（白色气泡，左对齐）
   ↓
5. AI 内容逐字流式显示 ▌
   ↓
6. 流式完成，光标消失
   ↓
7. 可以继续发送新消息
```

### 视觉反馈

- ✅ **输入框聚焦**: 边框变蓝
- ✅ **按钮悬停**: 透明度变化 (0.9)
- ✅ **按钮点击**: 轻微缩放 (0.98)
- ✅ **禁用状态**: 透明度降低 (0.6)
- ✅ **滚动条**: 自定义样式，8px 宽度

### 动画效果

- **新消息出现**: 平滑滚动到底部
- **流式文字**: 实时追加，自动滚动跟随
- **错误提示**: 淡入显示，可关闭

## 响应式设计

### 消息气泡

- **最大宽度**: 70% (防止过长消息占满屏幕)
- **自适应高度**: 根据内容自动调整
- **圆角**: 8px
- **阴影**: 轻微阴影效果

### 布局适配

- **最小宽度**: 建议 800px
- **最小高度**: 建议 600px
- **全屏支持**: 自动填充窗口

## 特殊状态

### 空状态（无消息）

```
        🤖
  开始与 AI 对话吧！
输入你的问题，按 Enter 发送
```

### 加载状态

- 输入框禁用
- 发送按钮显示"发送中..."
- 显示取消按钮
- AI 消息末尾显示光标 ▌

### 错误状态

```
┌─────────────────────────────────────────┐
│ ⚠️ 发送消息失败，请重试          [✕]   │
└─────────────────────────────────────────┘
```

## UI 元素详情

### 消息时间戳

- **格式**: HH:MM (24小时制)
- **位置**: 每条消息右上角
- **样式**: 小字号，半透明

### 角色标识

- **用户**: 👤 你
- **AI**: 🤖 AI

### 输入提示

- **占位符**: "输入消息... (Enter 发送, Shift+Enter 换行)"
- **字符计数**: 实时显示输入字符数
- **状态提示**: 根据状态显示不同文字

## 键盘快捷键

- `Enter` - 发送消息
- `Shift + Enter` - 换行
- `Ctrl/Cmd + K` - 清空输入框（可扩展）

## 可访问性

- ✅ **键盘导航**: 支持 Tab 键切换
- ✅ **按钮状态**: 清晰的禁用状态
- ✅ **错误提示**: 醒目的错误信息
- ✅ **加载反馈**: 明确的加载提示

## 性能优化

- ✅ **虚拟滚动**: 消息列表优化（可扩展）
- ✅ **防抖处理**: 输入框优化（可扩展）
- ✅ **内存管理**: 及时清理取消的请求
- ✅ **流式渲染**: 逐步渲染 AI 响应

---

这个界面设计注重用户体验，提供流畅的对话体验和清晰的视觉反馈！
